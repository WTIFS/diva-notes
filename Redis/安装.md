# Cluster
```bash
# 1. 安装 Redis

# 2. 复制仓库
git clone git@github.com:redis/redis.git

# 3. 修改配置
cd redis/utils/create-cluster
vim create-cluster

# 修改 BIN_PATH 为 redis-server 所在目录
BIN_PATH="/usr/local/bin/"

# 启动 redis-server
create-cluster start
# 建立集群
create-cluster create
```
